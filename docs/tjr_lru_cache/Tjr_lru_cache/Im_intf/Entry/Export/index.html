Module Entry.Export

type 'v entry =
| Insert of {
value : 'v;
dirty : Internal.dirty;
}
| Delete of {
dirty : Internal.dirty;
}
| Lower of 'v option

Cache map entries; values in the map are tagged with a last-accessed time and a dirty flag

Entries in the cache for key k:

  • Insert v (dirty=true/false)

    • this occurs on insert
  • Delete (dirty=true/false)

  • Lower vopt

    • this occurs when we check the lower layer for a non-existing entry in cache; if we find a value, we insert Lower (Some v), else Lower None; in either case, there is no need to do anything further (ie the entry is not dirty)

  • (No entry)

    • for a key that hasn't been seen before

Additionally, each entry has a last-accessed time
